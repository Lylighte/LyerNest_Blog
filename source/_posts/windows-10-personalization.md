### 彻底移除功能

Windows 10 会默认在任务栏推送*资讯与兴趣*，且该选项常驻任务栏设置。为彻底移除它：

1. 右键任务栏 - 资讯与兴趣 - 关闭。（若只是不想在桌面看到它，做完这步就行了）
2. 打开 `regedit`（注册表编辑器），再打开 `HKLM\Software\Policies\Microsoft\Windows\Windows Feeds`，在右面新建一个 DWORD（32位）键值（下称 DWORD），名称"EnableFeeds"数值为0，确认。如不存在相应的项（文件夹），则在左侧新建。
3. 继续打开 `HKCU\Software\Microsoft\Windows\CurrentVersion\Feeds`，在右面查看 DWORD 类型的键值，名称是"ShellFeedsTaskbarViewMode"，确认其数值为2。如没有，则新建之。
4. 重启文件资源管理器。

> 注：系统中本没有这个功能的，就不要凑热闹了；`HKLM` 实际显示为 `HKEY_LOCAL_MACHINE`，`HKCU` 实际显示为 `HKEY_CURRENT_USER`。参见：[分享彻底禁用win10任务栏资讯和兴趣功能的方法](https://bbs.pcbeta.com/viewthread-1944455-1-1.html)。

### 常规设置

#### 任务栏与开始菜单

任务栏设置（设置-个性化-任务栏）：
- **合并任务栏按钮：从不**
- 当我……替换为 Windows PowerShell：关
- 通知区域：始终显示所有图标

“开始”菜单设置（设置-个性化-开始）：
- 偶尔在“开始”菜单中显示建议：关
- ……显示最近打开的项：关
- 显示与账户相关的通知：关
- 全屏开始屏幕：视个人兴趣选择开启/关闭

桌面图标设置（设置-个性化-主题）：
- 右侧桌面图标设置：
  * 勾选需要的桌面图标
  * 取消勾选下方“允许主题更改桌面图标”

#### 设备

鼠标设置（设置-设备-鼠标）：
- 右侧其他鼠标选项：
  - 指针选项-提高指针精确度：关

> 注：这样取消的是*鼠标加速度*，光标移动速度与你移动鼠标的速度*线性相关*，在某些场合有效果。参见：[【玩竞技类游戏关闭这个选项可以大幅提高你的表现】](https://zhuanlan.zhihu.com/p/61945851)。

触摸板设置：
- 按需启用或关闭手势，详细配置略
- 三指单击建议设置为中键（鼠标按下滚轮对应的按键），有特别的用途，见后文

### 扩展设置

#### 日期和时间

区域设置（设置-时间和语言）
- 区域-国家和地区：按需要修改
注：这样会影响你在 Microsoft Store 等应用中看到的内容。
- 语言
  - 按需添加语言，例如添加 English(United States) 作为纯英文输入法。

> 注：玩过 Java 版 Minecraft 的都知道这意味着乱切输入法的概率大幅度降低。

- 在任务栏的时间窗格显示秒
  1. 首先确认你的系统是 Windows 10，然后在 `regedit` 中打开 `HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced`
  2. 在右侧新建一个 DWORD 值，并命名为 `ShowSecondsInSystemClock`，双击打开将数值数据修改为1，确定保存
  3. 重启文件资源管理器

> 注：Windows 11 右键任务栏时间，调整日期/时间，可见“显示秒数（耗电更多）”选项。
> > 微软的节能减排绿色建议也是一门生意。

### 一些快捷键

在系统中：

`Win` + `R`：运行
- 选定任务，按`Ctrl`+`Shift`+`Enter`，可指定“以管理员身份运行”

`Win` + `V`：**剪贴板**

`Win` + `E`：打开文件资源管理器（Explorer）

`Win` + `S`：打开搜索（Search）
- 事实上按下 `Win` 可以直接输入需要搜索的内容，自动弹出搜索框

`Win` + `P`：多屏幕切换
- 在笔记本电脑外接显示屏时有显著用途

`Win` + `Tab`：将所有窗口平铺显示

`Alt` + `Tab`：切换窗口

`Win` + `D`：**返回桌面**（Desktop）

`Win` + `空格`：切换输入法
- 卡输入法时用

在编辑器（Word，Visual Studio Code，……）中

`Ctrl` + `N`：在编辑器中新建窗口/文件（New）

`Ctrl` + `F`：**查找内容**（Find）

`Ctrl` + `A`：全选（All）

`Ctrl` + `D`：选中光标附近的词

`Ctrl` + `S`：保存（Save）
- **干活一定要记得保存与备份！**

`Shift` + `↑↓←→`：选中文本，取代鼠标拖动选取

鼠标中键（按下滚轮对应的按键）
- 对浏览器的选项卡直接中键，可关闭之
- 浏览器浏览网页中键，上下左右移动，可自动移动网页
- 对任务栏的应用中键，可开启一个相同的应用窗口
